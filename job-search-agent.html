🤖 Project

AI Job Search Agent

A live production agent that automates my weekly job market scan. Every Monday it sources fresh roles from across Europe, ranks them with Claude against my profile, and delivers a curated shortlist of the top 60 opportunities.

Live · Running weekly Anthropic API Adzuna API GitHub Actions Python
Why I built it

Solving my own friction

I was running a focused job search across Belgium, the Netherlands, Luxembourg, and the UK — and spending hours every week on generic job boards filtering noise to find the few roles that actually matched my profile.

So I built the agent I needed. One-time setup, weekly value forever. The pattern is the same one I use at work: identify the friction, build the agent, ship it.

Same pattern, different scale.
How it works

The weekly pipeline

01
Schedule GitHub Actions triggers the agent every Monday morning. No manual intervention, no missed weeks.
02
Source The Adzuna API pulls fresh job listings across my target geographies and role keywords — Programme Manager, Cloud Transformation, Delivery Lead, AI/ML.
03
Rank The Anthropic API (Claude) scores each role against my profile — career trajectory, certifications, geographic preferences, and the work I actually want to do next. Not keyword matching. Real reasoning.
04
Deliver The top 60 ranked opportunities land in a structured weekly digest, ready for me to review and apply.
